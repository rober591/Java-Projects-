This program is a Java Console Application called RockPaperScissors.RulesThe rules of the game are as follows:1. Each player chooses Rock, Paper, or Scissors.2. If both players choose the same thing, the round is a tie.3. Otherwise:a. Paper wraps Rock to winb. Scissors cut Paper to winc. Rock breaks Scissors to winThe program first asks the user how many rounds he/she wants to play.4. Maximum number of rounds = 10, minimum number of rounds = 1.Ê5.
